## What is this project?

This is an open-source toolkit and template vault for playing Solo RPGs using [Obsidian](https://obsidian.md). It supports journaling, tracking, map drawing, dice rolling, and more—all in one place.

Whether you're exploring the wastelands, delving into fantasy ruins, or building your own system, this vault gives you the structure and tools to stay immersed and organized.
